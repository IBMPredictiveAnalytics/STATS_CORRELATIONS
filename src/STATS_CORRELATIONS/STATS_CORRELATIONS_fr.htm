Corrélations bivariées avec intervalles de confiance

Cette procédure calcule les corrélations bivariées et les intervalles de confiance asymptotiques (Fisher). Les variables dotées d'un niveau de mesure nominal sont exclues.

Variables Choisissez les variables d'une matrice de corrélation carrée (la corrélation de chaque variable avec chacune des autres variables est générée).

Avec variables Si les variables sont placées dans cette liste, seules les corrélations des variables de la première liste avec les variables répertoriées ici sont générées. Par exemple, si la première liste contient cinq variables et la deuxième en contient deux, dix corrélations sont générées.

Niveau de confiance Entrez le niveau de confiance souhaité pour l'intervalle de confiance sous forme de pourcentage.

Estimation de l'intervalle de confiance Choisissez Fisher pour utiliser la transformation Fisher basée sur la normalité approximative de la transformation Z de la corrélation. Choisissez Bootstrap pour calculer les intervalles de confiance du bootstrap.

•Le bootstrap requiert l'option de bootstrap sous licence. Autrement, une erreur se produit.

Valeurs utilisateur manquantes Choisissez la suppression par paire pour maximiser l'utilisation des données ou la suppression liste par liste pour une base d'observations cohérente pour chaque corrélation. Vous pouvez également choisir de traiter les valeurs manquantes comme valides. Les valeurs manquantes du système (sysmis) sont toujours exclues.

SPLIT FILES est pris en charge.

Merci à Bruce Weaver pour son aide sur cette procédure.

Fonctions supplémentaires

Exécutez la commande
STATS CORRELATIONS /HELP.
ou, dans Statistics version 23 ou ultérieure, appuyez sur la touche F1 avec le curseur dans la commande pour afficher l'aide sur la syntaxe.

TO et ALL sont pris en charge dans les listes de variables de Statistics 23 (ou version ultérieure).

Conditions requises

Cette commande requiert le plug-in de programmabilité Python.
