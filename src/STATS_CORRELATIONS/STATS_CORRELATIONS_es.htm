Correlaciones bivariadas con intervalos de confianza

Este procedimiento calcula correlaciones bivariadas e intervalos de confianza asintóticos (Fisher). Las variables con un nivel de medición nominal se excluyen.

Variables Seleccione las variables para una matriz de correlaciones cuadrada, es decir, se produce la correlación de cada variable con todas las demás.

Con variables Si se incluyen variables en esta lista, entonces solo se producen las correlaciones de las variables de la primera lista con las variables incluidas en esta lista. Por ejemplo, si hay cinco variables en la primera lista y dos en la segunda, se producen diez correlaciones.

Nivel de confianza Introduzca al nivel de confianza deseado para el intervalo de confianza en forma de porcentaje.

Estimación del intervalo de confianza Seleccione Fisher para utilizar la transformación de Fisher en función de la normalidad aproximada de la transformación Z de la correlación. Seleccione Simulación de muestreo para calcular intervalos de confianza de simulación de muestreo.

•Para usar Simulación de muestreo es necesario disponer de licencia para la opción de simulación de muestreo. De lo contrario, se producirá un error.

Valores perdidos de usuario Seleccione la eliminación por parejas para maximizar el uso de los datos o la eliminación por lista para disponer de una base coherente para los casos para cada correlación. Como alternativa, puede optar por tratar los valores perdidos como válidos. Los valores perdidos del sistema (sysmis) siempre se excluyen.

Se admite SPLIT FILES.
