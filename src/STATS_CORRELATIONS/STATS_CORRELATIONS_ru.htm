Парные корреляции с доверительными интервалами

Эта процедура вычисляет двумерные корреляции и асимптотические доверительные интервалы (Фишера). Переменные с номинальным уровнем измерения исключаются.

Переменные Выберите переменные для квадратной корреляционной матрицы, то есть вычисления корреляции каждой переменной с каждой другой переменной.

С переменными Если переменные включены в этот список, выводятся только корреляции переменных из первого списка с перечисленными здесь переменными. Например, если в первом списке - пять переменных, а во втором - две, получится десять корреляций.

Доверительный уровень Введите требуемый доверительный уровень для доверительного интервала в процентах.

Оценка доверительных интервалов Выберите Фишер для использования преобразования Фишера на основе приблизительной нормальности Z-преобразования корреляции. Выберите Бутстреп для вычисления доверительных интервалов бутстрепа.

•Для бутстрепа требуется, чтобы была лицензирована опция бутстрепа. В противном случае произойдет ошибка.

Пользовательские значения отсутствия Выберите попарное исключение, чтобы максимизировать использование данных или удаление по списку для содержательных наблюдений на основе каждой корреляции. Можно также выбрать для обработки пропущенных значений как допустимых. Системные значения отсутствия (sysmis) всегда исключаются.

Поддерживается SPLIT FILES.

Дополнительные возможности

Введите команду
STATS CORRELATIONS /HELP.
или, в Statistics версии 23 или новее нажмите клавишу F1, когда указатель находится на команде, чтобы вывести справку по синтаксису.

TO и ALL поддерживаются в списках переменных в Statistics 23 или новее.
