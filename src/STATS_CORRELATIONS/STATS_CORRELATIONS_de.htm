Bivariate Korrelationen mit Konfidenzintervallen

Mit dieser Prozedur werden bivariate Korrelationen und asymptotische Konfidenzintervalle (Fisher) berechnet. Variablen mit einem nominalen Messniveau werden ausgeschlossen.

Variablen Wählen Sie die Variablen für eine quadratische Korrelationsmatrix aus, d. h. es wird eine Korrelation jeder Variablen mit jeder anderen hergestellt.

Mit Variable Werden Variablen in diese Liste aufgenommen, so werden nur die Korrelationen der Variablen in der ersten Liste mit den hier aufgeführten Variablen erstellt. Wenn zum Beispiel fünf Variablen in der ersten Liste und zwei in der zweiten Liste enthalten sind, ergeben sich zehn Korrelationen.

Konfidenzniveau Geben Sie das gewünschte Konfidenzniveau für das Konfidenzintervall in Prozent ein.

Konfidenzintervallschätzung Wählen Sie 'Fisher' aus, um die Fisher-Transformation zu verwenden, die auf der approximierten Normalität der Z-Transformation der Korrelation basiert. Wählen Sie 'Bootstrap' aus, um Bootstrap-Konfidenzintervalle zu berechnen.

•Bootstrap setzt voraus, dass die Bootstrap-Option lizenziert ist. Andernfalls tritt ein Fehler auf.

Benutzerdefiniert fehlende Werte Wählen Sie die paarweise Löschung, um die Nutzung der Daten zu maximieren, oder die listenweise Löschung, um eine einheitliche Fallbasis für jede Korrelation zu erhalten. Alternativ können Sie auch fehlende Werte als gültig behandeln. Systemdefiniert fehlende Werte (sysmis) werden immer ausgeschlossen.

SPLIT FILES wird unterstützt.

Vielen Dank an Bruce Weaver für seiner Unterstützung bei dieser Prozedur.

Weitere Funktionen

Führen Sie den folgenden Befehl aus:
STATS CORRELATIONS /HELP.
In Statistics ab Version 23 können Sie alternativ die Taste F1 drücken, wenn sich der Cursor auf dem Befehl befindet, um die Syntaxhilfe anzuzeigen.

TO und ALL werden in Variablenlisten in Statistics ab Version 23 unterstützt.

Anforderungen

Für diesen Befehl ist das Python Programmability Plug-in erforderlich.
