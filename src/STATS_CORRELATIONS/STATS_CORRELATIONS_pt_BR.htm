Correlações bivariadas com intervalos de confiança

Este procedimento calcula as correlações bivariadas e os intervalos de confiança assintóticos (Fisher). Variáveis com um nível de medição nominal são excluídas.

Variáveis: escolha as variáveis para uma matriz de correlação quadrada, ou seja, é produzida a correlação de cada variável com todas as outras.

Com variáveis: se forem colocadas variáveis nessa lista, somente as correlações das variáveis na primeira lista com as variáveis listadas aqui serão produzidas. Por exemplo, se houver cinco variáveis na primeira lista e duas na segunda, dez correlações serão produzidas.

Nível de confiança: insira o nível de confiança desejado para o intervalo de confiança como uma porcentagem.

Estimação de intervalo de confiança: escolha Fisher para usar a transformação Fisher baseada na normalidade aproximada da transformação Z da correlação. Escolha Autoinicialização para calcular intervalos de confiança de autoinicialização.

•A autoinicialização requer que a opção de autoinicialização seja licenciada. Caso contrário, ocorrerá um erro.

Valores ausentes do usuário: escolha Exclusão pairwise para maximizar o uso dos dados ou Exclusão listwise para uma base de caso consistente para cada correlação. Como alternativa, é possível optar por tratar os valores ausentes como válidos. Os valores ausentes do sistema (sysmis) são sempre excluídos.

SPLIT FILES é suportado.

Recursos adicionais

Execute o comando
STATS CORRELATIONS /HELP.
ou, em Statistics versão 23 ou posterior, pressione F1 com o cursor no comando para exibir a ajuda de sintaxe.

TO e ALL são suportados nas listas de variáveis em Statistics 23 ou posterior.
